# Starter Frontend — Optical Shop OS

This is a minimal React + Vite frontend that demonstrates logging in and using the protected `/customers` API from the starter backend.

Quick start

```bash
cd "C:/Users/admin/Desktop/Opticals solution/starter-frontend"
npm install
npm run dev
```

Usage
- Open http://localhost:5173 (Vite default)
- Login with the backend default admin: `admin` / `admin123`

Notes
- Backend must be running at `http://localhost:3000` and CORS must be enabled (the starter backend includes CORS middleware).
